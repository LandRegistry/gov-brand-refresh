GOV.UK Brand Refresh

GOV.UK is undergoing a rebrand launching with the GOVUK app on the 25th June. The GOV.UK website and Government Services running there must also change on the same day to the new brand. How does this effect us and what do we need to do?

The first thing to do is populate the List of services and webpages spreadsheet, in particular filling out the URL column. If there are any services, products or webpages missing please add them. With this information we can ensure each service gets the correct advice and support they need in making the design changes, particularly our external facing services that require our own branding.

Next, there is a guide you can use to identify where your service sits and what changes you may need to make.

Service, product or webpage using a GDS domain

For example Search for Land and Property Information

These can be identified by:

If the url for the webpage is in the style of gov.uk/myservice. In this instance you may not need to do anything and the header and footer will update automatically. Email the URL to Grace Buttall-Jensen. and I will be able to confirm.

If the url ends with .service.gov.uk. In this instance service teams will need to update and apply the new branding styles. We will also need to ensure that anything in the header finds its new correct home, like the service name in the navigation bar.

We are currently putting together prototypes to illustrate the required design changes.

NOTE - this is not for production code. Service/products must use the GOV.UK Frontend.

Changes must be made for 25th June

All other domains, external customer facing and internal

For example Mantain LLC, DRS, Application Processing

We will want to make design changes to our header and footer to keep a consistent feel across government services. This may have a knock on effect to our navigation and other components such as phase banners, Welsh language toggle, etc. These service want to look like GOV.UK but must not use their branding.

We are currently putting together prototypes to illustrate the required design changes for external and internal facing services .

NOTE - this is not for production code. Service/products must use the GOV.UK Frontend but remove GOV.UK branding and font.

Changes should be made from 25th June in a reasonable timeframe with our external facing services taking priority.
