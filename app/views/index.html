GOV.UK Brand Refresh

GOV.UK is undergoing a rebrand launching with the GOVUK app on the 25th June. The GOV.UK website and Government Services running there must also change on the same day to the new brand. How does this effect us?

If the service, product or webpage uses a GDS domain

These can be identified by having one of the following in the URL:

  • gov.uk/myservice
  • .service.gov.uk

For example Search for Land and Property Information

We are currently putting together prototypes to illustrate the required design changes.

NOTE - this is not for production code. Service/products must use the GOV.UK Frontend.

Changes must be made for 25th June

All other domains

For example DRS, Application Processing

We are currently putting together prototypes to illustrate the required design changes for external and internal facing services .

NOTE - this is not for production code. Service/products must use the GOV.UK Frontend but remove GOV.UK branding and font.

Changes should be made from 25th June in a reasonable timeframe.

Additional guidance

For more information on Making your service look like GOV.UK including instructions on when your service isn't on GOV.UK.

If you are unsure of what branding or design changes your service/product needs please email Grace Buttall-Jensen.
